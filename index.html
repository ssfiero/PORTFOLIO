Professional Project

Built with: HTML5, CSS3, Twitter Bootstrap, Google Docs

Summary:

This project was from an actual client, that I worked on with my classmates, with our teacher overseeing. I experienced the project from conception to delivery.

The website includes six buttons that link to six different forms that can be filled out based on the customer's needs. The client specifically requested six different forms.
